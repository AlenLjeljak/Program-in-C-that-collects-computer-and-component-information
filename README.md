# Program-in-C-that-collects-computer-and-component-information
This is a program made in C# that reviews and collect computer and component information. 
The informations included in this program are:

Operating system and Windows version
Any network information (network adapters and IP addresses)
Hardware
Motherboard
Processor
Graphics card
RAM-modules
SSD-module
Battery
BIOS
Uptime
Windows Defender
Latest Windows Update
